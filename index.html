TermuxInfo

Termux - это эмулятор терминала для Android, включающий большую коллекцию пакетов Linux.

Системные требования

Android 5.0 - 12.0
Процессоры: AArch64, ARM, i686, x86_64.
Не менее 300 МБ дискового пространства.

Полезные пакеты

Termux API

Предоставляет функциональные возможности устройства в виде API
Установка пакета: pkg install termux-api

Команды:

termux-battery-status
Получите состояние аккумулятора устройства.
termux-brightness
Установите яркость экрана от 0 до 255.
termux-call-log
Список журнала вызовов.
termux-camera-info
Получите информацию о камерах устройств.
termux-camera-photo
Сделайте фотографию и сохраните ее в файл в формате JPEG.
termux-clipboard-get
Получите текст системного буфера обмена.
termux-clipboard-set
Задайте текст системного буфера обмена.
termux-contact-list
Перечислите все контакты.
termux-dialog
Отображение диалогового окна ввода текста.
termux-download
Загрузите ресурс с помощью системного менеджера загрузок.
termux-fingerprint
Используйте датчик отпечатков пальцев на устройстве для проверки подлинности.
termux-infrared frequencies
Запросите поддерживаемые несущие частоты инфракрасного передатчика.
termux-infrared-transmission
Передача инфракрасного рисунка.
termux-job-scheduler
Запланируйте выполнение сценария Termux позже или периодически.
termux-location
Получите местоположение устройства.
termux-media-player
Воспроизведение медиафайлов.
termux-media-scan
Интерфейс MediaScanner, сделайте изменения файлов видимыми в галерее Android
termux-microphone-recording
Запись с помощью микрофона на устройстве.
termux-уведомление
Отображение системного уведомления.
termux-notification-remove
Удалите уведомление, ранее показанное с помощью termux-notification --id.
termux-sensor
Получайте информацию о типах датчиков, а также данные в реальном времени.
termux-share
Предоставьте общий доступ к файлу, указанному в качестве аргумента, или к тексту, полученному на stdin.
termux-sms-list
Список SMS-сообщений.
termux-sms-send
Отправьте SMS-сообщение на указанный номер (номера) получателя.
termux-storage-get
Запросить файл из системы и вывести его в указанный файл.
termux-телефония-call
Позвоните на номер телефона.
termux-телефония-cellinfo
Получайте информацию обо всей наблюдаемой информации о клетках от всех радиостанций на устройстве, включая первичные и соседние ячейки.
termux-telephony-deviceinfo
Получите сведения об устройстве телефонии.
termux-toast
Отображение временного всплывающего уведомления.
termux-torch
Переключатель светодиодной горелки на устройстве.
termux-tts-engines
Получите сведения о доступных механизмах преобразования текста в речь.
termux-tts-speak
Произнесение текста с помощью системного механизма преобразования текста в речь.
termux-usb
Список USB-устройств или доступ к ним.
termux-vibration
Вибрируйте устройство.
termux-volume
Изменение громкости аудиопотока.
termux-wallpaper
Измените обои на устройстве.
termux-wifi-connectioninfo
Получите информацию о текущем подключении Wi-Fi.
termux-wifi-enable
Включите/выключите Wi-Fi.
termux-wifi-scaninfo
Получите информацию о последнем сканировании Wi-Fi.

Ngrok

Пакет для создания веб-сервера
Установка пакета: pkg install ngrok

Команды

./ngrok - первый запуск
./ngrok http 80 - запуск сервера

Apache2

Пакет для собздания веб-сервера
Установка пакета: pkg install apache2

Команды

apachectl - запуск сервера
apachectl -k stop - остановка сервера

OpenSSH

Пакет для для удаленного входа в систему по протоколу SSH
Установка пакета: pkg install openssh

Команды

sshd - запуск OPENSSH сервера
pkill sshd - остановка
passwd - установка нового пароля

Nano

Пакет для редактирования текстовых файлов
Установка пакета: pkg install Nano

Команды

nano *путь к файлу* - открытие файла
